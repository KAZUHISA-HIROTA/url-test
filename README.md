# 🧪 HTML表示テスト用リポジトリ

> ようこそ。ここは「HTMLファイルをWebで公開するとどう見えるか」を試すための実験場です。

---

## 🎯 このリポジトリの目的

GitHub Pages の仕組みを使って、HTMLファイルを誰でもブラウザで閲覧できる形で公開するテストを行っています。

「リポジトリ画面（github.com）に置かれたHTML」と、「Pagesで公開されたHTML」が**全く違う見え方をする**ことを体感するのが目的です。

---

## 🚀 表示テスト

### 👇 こちらをクリック

### **[📊 パレット統計を見る](https://kazuhisa-hirota.github.io/url-test/AdominDocs/パレット統計.html)**

クリックすると、デザインされた状態のHTMLが表示されます。

---

## 🔍 ちょっとした実験

同じファイルを、2つの違う見方で開いてみてください。

| アクセス方法 | URL | 見え方 |
|---|---|---|
| 😎 **Pages経由（正解）** | [github.io/...](https://kazuhisa-hirota.github.io/url-test/AdominDocs/パレット統計.html) | デザインされた資料 |
| 🤔 **リポジトリ経由（残念）** | [github.com/...](https://github.com/KAZUHISA-HIROTA/url-test/blob/main/AdominDocs/パレット統計.html) | HTMLのソースコード |

**同じファイルなのに、URLが違うだけで見え方が変わる**のがGitHub Pagesの面白いところ。

---

## 🗂 リポジトリ構成

```
url-test/
├── README.md              ← 今見ているコレ
└── AdominDocs/            ← HTMLファイルの保管場所
    └── パレット統計.html    ← テスト用資料
```

---

## 💡 豆知識

- ファイルを更新すると、**1〜3分後に自動で公開サイトに反映**されます
- ブラウザに古いキャッシュが残ることがあるので、表示確認は `Ctrl + F5` でスーパーリロードがおすすめ
- URLは大文字小文字を区別します（タイプミス注意⚠️）
